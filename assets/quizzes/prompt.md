以下の形式で、指定された年に関するクイズを作成してください。各クイズは次の情報を含む必要があります：

1. 質問（question）
2. 選択肢（options）: 少なくとも4つの選択肢
3. 正解の選択肢（answer）
4. 解説（explanation）
5. タグ（tags）: 質問のカテゴリに関連するタグ

クイズ形式:
[
    {
        "question": "<質問文>",
        "options": [
            "<選択肢1>",
            "<選択肢2>",
            "<選択肢3>",
            "<選択肢4>"
        ],
        "answer": "<正解の選択肢>",
        "explanation": "<解説>",
        "tags": [
            "<タグ1>",
            "<タグ2>",
            "<タグ3>"
        ]
    },
    ...
]
