以下の形式で、指定された年に関するクイズを作成してください。各クイズは次の情報を含む必要があります：

1. 質問（question）
2. 選択肢（options）: 少なくとも 4 つの選択肢
3. 正解の選択肢（answer）
4. 解説（explanation）
5. タグ（tags）: 質問のカテゴリに関連するタグ

クイズ形式:
[
{
"question": "<質問文>",
"options": [
"<選択肢 1>",
"<選択肢 2>",
"<選択肢 3>",
"<選択肢 4>"
],
"answer": "<正解の選択肢>",
"explanation": "<解説>",
"tags": [
"<タグ 1>",
"<タグ 2>",
"<タグ 3>"
]
},
...
]

2019 年に関するクイズを作成してください。
